# MiseNote（試作品）

飲食店の日報、売上、在庫をひとつの画面で管理するためのWebアプリ試作品です。

## できること

- 今日の売上・客数・客単価・在庫アラートを確認
- 日報の入力と保存
- 支払い方法ごとの売上入力と自動集計
- 食材の追加、検索、絞り込み、数量変更
- 入力内容をブラウザー内に自動保存
- スマートフォン用の表示
- iPhoneのホーム画面からアプリのように起動
- 一度開いたあとは、電波が弱い場所でも画面を起動

## 起動方法

いちばん簡単な確認方法は、`index.html` をダブルクリックすることです。追加のインストールは必要ありません。

Node.jsが入っている開発環境では、次の方法でも起動できます。

```powershell
node server.mjs
```

その後、ブラウザーで `http://127.0.0.1:4173` を開きます。

入力したデータは、そのブラウザーの中だけに保存されます。試作品のため、別のパソコンとの共有やログイン機能はまだありません。

## iPhoneで使う方法

1. 公開用URLをiPhoneのSafariで開きます。
2. Safariの共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。
4. 「Webアプリとして開く」をオンにして「追加」を押します。

ホーム画面にMiseNoteのアイコンが追加されます。

入力データはiPhone内に保存されます。パソコンとの自動共有や複数人での同時利用は、ログインとデータベースを追加する次の開発段階で対応します。

## ファイル

- `index.html`：画面の骨組み
- `styles.css`：色や配置などの見た目
- `app.js`：入力、保存、集計などの動き
- `server.mjs`：このパソコン内だけで試作品を表示する補助役
- `manifest.webmanifest`：iPhoneにアプリ名やアイコンを伝える設定
- `service-worker.js`：一度開いた画面を端末に保存する仕組み
